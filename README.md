Home Of News is a fast, responsive, and user-friendly news website built using HTML, CSS, and JavaScript. It fetches real-time news from a live news API and presents articles in a clean, modern interface optimized for both desktop and mobile devices.

The website features infinite scrolling for seamless news browsing, live search to quickly find articles, and article detail views that open in the same tab for a smooth reading experience. A built-in dark mode with eye-friendly colors improves readability in low-light conditions, while modern UI elements and animations enhance user experience.
